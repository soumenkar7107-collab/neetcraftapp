# NEETCraft — Premium NEET Prep System

A full-featured, gamified NEET preparation dashboard built with Next.js 14.

## Features
- XP + Level system (gain XP by completing tasks, chapters, logging study time)
- Daily progress tracking (Questions + Study Time vs goals)
- Full NEET syllabus tracker (Physics, Chemistry, Biology — all chapters)
- Task manager with deadlines + XP rewards
- Test score logger with Recharts line graphs
- Spaced repetition revision planner (Day 1/3/7/14)
- Pomodoro focus timer
- AI Coach with personalized daily feedback
- 100% localStorage — no backend needed

## Deploy to Vercel

### Step 1 — Push to GitHub
```bash
git init
git add .
git commit -m "Initial NEETCraft commit"
git remote add origin https://github.com/YOUR_USERNAME/neetcraft.git
git push -u origin main
```

### Step 2 — Deploy on Vercel
1. Go to https://vercel.com
2. Click "Add New Project"
3. Import your GitHub repo
4. Click Deploy — done!

## Run Locally
```bash
npm install
npm run dev
```
Open http://localhost:3000
